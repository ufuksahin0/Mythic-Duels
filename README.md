Mythic Duels
---------------------------

Description:
This is my project for the Java Card Game. It is a strategy card game where you play against the computer.
I used Java Swing for the GUI and standard OOP principles like Inheritance and Polymorphism.

How to Run:
1. Open the project in IntelliJ IDEA.
2. Run 'Main.java' in the 'com.cardgame.main' package.
3. The game menu should appear.

Features:
- Play against an AI (Computer Player).
- Cards have Attack, Defense, and Cost.
- High Score system is implemented (saves to file).
- Save/Load game functionality works.
- Custom images for cards (I used placeholders/generated ones if images are missing).

Notes:
- The AI tries to counter your moves but sometimes plays aggressively.
- If the images don't load, make sure the 'resources' folder is in the right place.

Enjoy!
